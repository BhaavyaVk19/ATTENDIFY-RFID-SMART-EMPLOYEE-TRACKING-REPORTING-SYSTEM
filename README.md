ATTENDIFY-RFID is an RFID-based employee attendance and monitoring system designed to automate workforce tracking. The system enables employees to mark attendance by simply scanning their RFID cards at entry and exit points. Attendance records are processed by an LPC2148 microcontroller and transmitted to a Linux-based application for storage, management, and report generation.

The project eliminates manual attendance processes, improves accuracy, reduces administrative effort, and provides real-time employee tracking and reporting.

🎯 Objectives

  Automate employee attendance management.
  Reduce manual attendance errors.
  Track employee entry and exit timings.
  Calculate working hours automatically.
  Generate attendance reports and activity logs.
  Provide user management through an admin interface.
